PocketPy 是一个轻量级的Python解释器，为嵌入至游戏引擎而设计。

它包含一个编译器和基于字节码的虚拟机，以及交互式命令窗的实现。所有功能均集成在单个头文件`pocketpy.h`中，不包含外部依赖项，能很方便地嵌入至你的应用。

请查阅 https://pocketpy.dev 获取更多细节，或者尝试 [在线Demo](https://blueloveth.github.io/pocketpy)。
